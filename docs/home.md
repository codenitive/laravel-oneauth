OneAuth Bundle
==============

OAuth and OAuth2 Auth bundle for Laravel

*OpenID support is currently under plan.*

## Installation

### Installation with Laravel Artisan

	php artisan bundle:install oneauth
	
### Bundle Registration

	'oneauth' => array('auto' => true),

### Run the migration

	php artisan migrate
